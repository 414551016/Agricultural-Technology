# 牛奶果幼苗培育（繁殖）
- [實生繁殖](https://github.com/414551016/Agricultural-Technology/blob/main/Chrysophyllum-Cainito.md#%E5%AF%A6%E7%94%9F%E7%B9%81%E6%AE%96)

牛奶果主要有兩大繁殖方式：實生繁殖與無性繁殖。實生苗多用來當砧木；若要保留母樹果實品質、提早結果，則以嫁接、高壓、扦插較適合。台灣資料指出，實生苗播後約 14～40 天萌芽，長到 4～5 片葉可移盆，約 6～8 個月後可嫁接。
原因是：種子苗可能變異大、結果慢；美國佛州 IFAS 資料也指出，星蘋果可用種子、嫁接、芽接與高壓繁殖，實生樹約 5～10 年結果，而無性繁殖苗可能 1～2 年開始結果。
```
實務上最建議的路線是：
種子育砧木 → 6～8 個月後嫁接優良品種 → 培育健壯嫁接苗 → 定植。
```
#### 我的建議育苗模式
您是要在田間種植，我建議採這個流程：
```
第 1 階段：播種育砧木
新鮮種子洗淨後立即播種，半陰環境育苗。

第 2 階段：移盆養根
4～5 片葉移盆，培養直根與側根，避免盆內盤根。

第 3 階段：6～8 個月嫁接
選擇果實大、甜度高、產量穩定的母樹接穗。

第 4 階段：嫁接苗強化
嫁接成活後逐步增加日照，培養主幹與 3～5 枝主枝。

第 5 階段：雨季前或春季定植
選排水良好位置，避免低窪積水，初期設支柱與覆蓋防草。

總結來說，牛奶果可以用種子繁殖，但若目標是穩定結果、縮短結果年限、保持優良果實品質，應以實生砧木＋嫁接苗為最佳方式。
```

## 實生繁殖
### 種子育苗流程
- 選果與取種：
  選擇完全成熟、果形端正、甜度高、無病蟲害的牛奶果作為採種來源。取出種子後，要把果肉與黏液洗乾淨，避免發霉。牛奶果種子不宜久放，台灣資料建議取出後立即播於苗床。
- 播種介質：
  育苗介質重點是：疏鬆、保水但不積水、排水良好、微酸至中性。牛奶果對土壤適應性廣，砂質土、黏質壤土都可，但以土層深厚、肥沃、排水良好處較佳。
  可用這種配方：腐熟有機質 30%＋壤土或培養土 40%＋粗砂／珍珠石 20%＋椰纖或稻殼炭 10%。
  重點不是肥越多越好，而是讓幼根能順利呼吸、伸展，避免悶根。
- 播種方式：
  幼苗期要半日照或遮蔭 40～60%，不可一開始就強烈曝曬。資料也指出苗期需適度遮蔭、勤澆水。
  澆水原則是：表土乾再澆、澆則澆透、不讓盆底積水。若水分過多，幼苗容易爛根；若忽乾忽濕，則新根發育慢，葉片容易黃化或停滯。
### 移盆與幼苗培育
當幼苗長到 4～5 片真葉，就可移到小盆或 3～5 吋育苗袋。移盆時不要硬拉苗，應帶土移植，避免主根受傷。資料指出長到 4～5 片葉後再移盆，6～8 個月後即可嫁接。
移盆後 7～10 天先放在半陰處，恢復後再逐步增加日照。牛奶果幼樹忌強烈直射與冷風，台灣農業知識入口網也提到幼樹忌陽光直射，應適當澆水。

## 無性繁殖
### 嫁接繁殖：最適合經濟栽培
若目標是未來穩定結果、果實品質一致，建議採嫁接苗。實生苗主要當砧木，等砧木莖粗約鉛筆粗、苗齡約 6～8 個月時，可以接上優良母樹枝條。台灣資料指出，嫁接苗能保持品種特性，且種植後約 1 年可開始結果，但山欖科種屬間嫁接可能有不親和問題。
嫁接注意事項：
- 砧木要健康：根系白、無根瘤、無爛根。
- 接穗要成熟但不老化：選半木質化枝條，芽點飽滿。
- 同種嫁接較安全：避免亂用不同山欖科植物作砧木，以免不親和。
- 嫁接後保濕遮陰：約 2～4 週觀察芽點是否萌動。
- 成活後逐步拆袋、除砧芽：避免砧木芽搶養分。

### 高壓與扦插
高壓可保留母樹性狀，但台灣資料指出高壓苗約需 6～7 個月才可成苗；扦插苗雖容易生根，但抗風性較差。
因此如果是家庭少量繁殖，高壓可試；若是要做田間栽培，仍以嫁接苗較理想。IFAS 的熱帶果樹繁殖資料也指出，高壓是讓枝條仍連在母樹時先長根，通常成功率較高，星蘋果類也常用高壓繁殖。
