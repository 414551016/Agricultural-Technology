# 黃金果幼苗培育
## 實生法（砧木）
黃金果（Abiu，Pouteria caimito）從種子開始育苗的第一原則是：種子越新鮮越好，不能乾放太久。 UF/IFAS 指出黃金果種子從果實取出後，通常只能維持幾天活性，應盡快種入乾淨、排水良好的介質；黃金果適合溫暖潮濕、避風、無霜害的環境，約 20–35°C 生長較佳。 台灣已有商業栽培，且被認為相當適應台灣中南部氣候，但在花蓮東部育苗時要特別注意冬季風、排水與苗期防寒。

# 問題：
### 黃金果育苗，可否使用扦插方法育苗？
可以試驗扦插，但我不建議把它當成黃金果主要育苗方式。對黃金果而言，較可靠的繁殖路線仍是：
```
實生苗育砧木 → 再嫁接優良品種。
```
UF/IFAS 的黃金果栽培資料指出，黃金果最常用種子繁殖，也可將優良品種嫁接或芽接在實生砧木上；嫁接苗通常可比實生苗更早結果。 另一篇菲律賓黃金果研究[AskIFAS](https://ask.ifas.ufl.edu/publication/HS300)也顯示，黃金果以黃金果實生苗作砧木進行靠接與劈接可以成功，但其他砧木親和性較差。
