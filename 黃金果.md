## 實生法（砧木）種植（較佳）：
黃金果（Abiu，Pouteria caimito）從種子開始育苗的第一原則是：種子越新鮮越好，不能乾放太久。 UF/IFAS 指出黃金果種子從果實取出後，通常只能維持幾天活性，應盡快種入乾淨、排水良好的介質；黃金果適合溫暖潮濕、避風、無霜害的環境，約 20–35°C 生長較佳。 台灣已有商業栽培，且被認為相當適應台灣中南部氣候，但在花蓮東部育苗時要特別注意冬季風、排水與苗期防寒。
#### 收到種子後，第一天就要做的事
選種：新鮮完全成熟果實取出的種子，不要乾放後的種子，這會影響發芽率，大幅降低。
若你的計畫是：種子育苗 → 養成實生砧木 → 日後嫁接優良母樹接穗
黃金果可用種子繁殖，但 UF/IFAS 也指出黃金果可嫁接或芽接於實生砧木上，嫁接苗通常比實生苗更早結果。
| 選種重點    | 原因                                                         |
| ------- | ---------------------------------------------------------- |
| 新鮮濕種子   | 黃金果種子離果後活性只能維持很短時間，應盡快播種。([Ask IFAS - Powered by EDIS][1]) |
| 台灣本地母樹  | 對台灣濕熱、雨季、土壤環境較有參考價值                                        |
| 母樹健康強健  | 砧木重點是根系、苗勢與抗逆性                                             |
| 種子飽滿大粒  | 通常苗勢較好                                                     |
| 不買乾燥舊種子 | 發芽率風險高                                                     |



先把種子外面的果肉洗乾淨，因為殘留果肉很容易發霉。洗好後不要曬乾，只要用紙巾稍微吸乾表面水分即可。若種子已經很乾、皺縮、變輕，發芽率通常會明顯下降。
建議你把種子分成兩批：
一批直接播在育苗盆；另一批用濕紙巾催芽，放在透明盒或夾鏈袋內觀察。這樣可以提高成功率，也方便判斷種子是否還有活性。

#### 育苗介質這樣配
黃金果幼苗最怕「悶、爛、積水」，所以不要直接用田土或黏重土。建議使用：
介質要做到「保濕但排水快」。黃金果喜濕潤，但忌乾旱與積水，水分管理不良會影響生長。 土壤方面，UF/IFAS 建議以肥沃、排水良好、pH 約 5.5–7.5 的土壤較適合。
```
椰纖土或泥炭土 50%＋珍珠石或粗砂 30%＋腐熟有機質 20%
```

#### 播種方法
用深一點的育苗盆或黑軟盆比較好，因為黃金果幼苗會長主根。每盆放一粒種子即可。
- 發芽後種法如下：
  - 種子平放或側放，覆土約 1–2 公分。
  - 不要埋太深，否則容易悶爛；也不要露在表面，否則容易乾掉。播完後澆透水一次，之後保持濕潤即可。
  - 放置位置以明亮散射光、50% 遮陰較適合，不要一開始就全日照曝曬。花蓮若風大，育苗期要放在有防風網或牆邊避風處。

#### 發芽期間管理
- 發芽適溫大約抓 25–30°C 最穩定。夏季、初秋最容易成功；冬季低溫時發芽會變慢，甚至爛種。
- 每天檢查介質濕度，但不要每天大量澆水。正確判斷方式是：手指摸表土下 1 公分，若仍濕就不用澆；若微乾再補水。盆底不可積水。
- 正常情況下，新鮮種子可能在數週內開始發根、出芽；若超過 6–8 週仍無動靜，可以輕輕翻查一粒，看是否已腐爛或仍有白色根點。

#### 幼苗長出後怎麼照顧
幼苗剛出土時先不要施肥。等長出 2–3 對真葉後，再開始很淡的肥培管理。
- 建議方式：
  - 第一個月：只澆水，不施肥。
  - 第二個月起：每 2–3 週給一次很淡的液肥，濃度約一般建議量的 1/4。
  - 三個月後：若苗勢穩定，可以少量補充有機質或緩效肥。
  - 不要一開始就重肥，否則容易傷根、葉尖焦枯。

#### 何時換盆？
- 當苗高約 15–25 公分、根系已抓住土團時，就可以換到較大的盆或育苗袋。換盆時要保留原土團，不要把根洗開，因為黃金果幼苗根系受傷後恢復較慢。
- 換盆後先放遮陰處 7–10 天，再慢慢增加日照。等苗高約 40–60 公分、莖幹較硬、葉片健康時，才適合準備定植到田間。

#### 田間定植前的重點
- 黃金果最後要種在排水良好、不長期積水、避強風、日照充足的位置。若是在花蓮新城一帶，建議幼苗期先不要種在完全空曠迎風處，可先用防風網、香蕉、木麻黃、防風林或臨時支架保護。
- 定植時不要種太深，原盆土表面與田土表面齊平即可。種太深容易悶根、爛根。

#### 最容易失敗的原因
- 種子太乾或放太久：黃金果種子不耐儲藏，買到乾種子時發芽率可能很低。
- 介質太黏、太濕：最常造成爛種。
- 一開始就全日照：幼苗葉片容易焦傷。
- 太早重肥：幼根容易受傷。
- 冬季低溫與強風：苗期生長停滯，嚴重時落葉或死亡。

# 嫁接：
- [黃金果實生幼苗長到何時可以嫁接？](https://github.com/414551016/Agricultural-Technology/blob/main/%E9%BB%83%E9%87%91%E6%9E%9C.md#%E9%BB%83%E9%87%91%E6%9E%9C%E5%AF%A6%E7%94%9F%E5%B9%BC%E8%8B%97%E9%95%B7%E5%88%B0%E4%BD%95%E6%99%82%E5%8F%AF%E4%BB%A5%E5%AB%81%E6%8E%A5)
## 黃金果實生幼苗長到何時可以嫁接？
  實生法培育的黃金果苗，通常等砧木「莖徑接近鉛筆粗、苗勢穩定、正在旺盛生長」時再嫁接最合適。 實務上多落在播種後約 6–12 個月，但不要只看月齡，要看苗本身的成熟度。
  - 建議符合以下條件再嫁接：
    | 判斷項目   | 適合嫁接標準             |
    | ------ | ------------------ |
    | 苗齡     | 約 6–12 個月以上        |
    | 苗高     | 約 40–80 公分         |
    | 嫁接位置莖徑 | 約 0.7–1.2 公分，接近鉛筆粗 |
    | 莖幹狀態   | 半木質化，不是嫩綠軟枝        |
    | 葉片狀態   | 葉色正常、無病蟲害、正在生長     |
    | 根系狀態   | 根系飽滿但未嚴重盤根         |
    | 氣候     | 溫暖、濕度穩定、避開寒流與連續豪雨  |

黃金果可用實生苗作砧木，再以優良母樹枝條作接穗嫁接；UF/IFAS 指出黃金果可嫁接或芽接於實生砧木上，嫁接苗有機會比一般實生苗更早結果，約 1–2 年進入結果。 另一篇菲律賓黃金果研究也指出，黃金果以黃金果實生苗作砧木進行靠接與劈接可成功繁殖，但以蛋黃果／仙桃類砧木效果不佳，表示砧穗親和性很重要。

### 最適合的嫁接季節：
  以台灣花蓮、東部環境來看，建議選：
  不建議在冬季嫁接，因為黃金果喜溫暖潮濕環境，適溫約 20–35°C，且幼樹對低溫與強風敏感。 UF/IFAS 的熱帶果樹繁殖資料也列出黃金果適合在溫暖月份進行劈接或貼接／腹接類嫁接。
- 春末到初夏：4–6 月。
  這是最推薦的時期，氣溫穩定、苗木開始旺盛生長，癒合速度較快。
- 初秋：9–10 月上旬。也可以，但要避開颱風、連續豪雨與東北季風轉強。

### 建議使用哪一種嫁接法？
對一般農民或初學者，我建議優先用：
1. 劈接：適合砧木與接穗粗細接近時使用。做法是把砧木上端平切，再由中央劈開，接穗削成楔形插入。這種方式簡單、容易對準形成層，適合黃金果小苗。
2. 腹接／貼接：適合砧木比接穗稍粗時使用。優點是砧木原本上部可暫時保留，嫁接失敗時砧木不會整株報廢。
3. 靠接：成功率通常較高，但操作慢、占空間，適合少量珍貴品種保存，不適合大量育苗。

### 接穗要怎麼選？
接穗是成功關鍵。請從已結果、品質穩定、果形漂亮、甜度佳、少乳汁、無病蟲害的母樹上剪取。
- 好的接穗條件：
  - 枝條約一年生或半成熟枝；
  - 枝皮轉褐綠色，不是太嫩的新梢；
  - 芽點飽滿，但還沒抽新芽；
  - 粗細接近砧木；
  - 長度約 8–12 公分；
  - 保留 2–3 個芽即可；
  - 葉片剪除，只留短葉柄，減少蒸散。
- 接穗剪下後最好當天使用。若不能馬上嫁接，可用微濕紙巾包住，放入塑膠袋，置於陰涼處或冰箱蔬果層短暫保存，但不要泡水、不要曬乾。

### 嫁接時最要注意的問題
1. 砧木與接穗要同種或近親和。
   黃金果最好用黃金果實生苗當砧木，可成功劈接與靠接。
3. 形成層一定要對準。
   嫁接不是把木頭插在一起，而是要讓砧木與接穗的形成層貼合。夏威夷大學熱帶果樹繁殖指南指出，嫁接成功需要接穗與砧木的形成層緊密接觸，才能重新連接木質部與韌皮部。
   若砧木比接穗粗，不必兩邊都對齊，至少要讓一側形成層準確貼合。
5. 切口要平整、乾淨、快速。
   刀子要利，切口不能毛。操作前可用酒精消毒刀具。切開後要盡快接合，避免切面氧化或流出乳汁後乾掉。
   黃金果屬山欖科，切口可能有乳汁。熱帶果樹繁殖指南提醒，會流乳汁的果樹有時較難嫁接，因乳汁可能妨礙砧穗直接接觸。 所以嫁接時動作要快，切面若乳汁太多，可稍微擦除後立即貼合包紮。
7. 包紮要密合，但不能勒死。
   嫁接帶要把切口完全包住，防止失水與雨水進入。接穗上端也可以用嫁接膜或塑膠袋保濕。包紮太鬆會乾掉，太緊則後期會勒傷。
9. 嫁接後避免日曬與雨淋。
   嫁接後 2–4 週內，最怕：
   - 強日照曬乾接穗；
   - 雨水進入切口；
   - 風吹造成接合處晃動；
   - 砧木萌芽搶養分；
   - 螞蟻、介殼蟲、薊馬危害嫩芽。
   - 建議放在 50% 遮陰、通風、有濕度但不悶熱的位置。田間嫁接則要用遮光網與透明袋保護，但袋內不能積水悶爛。

### 如何提高嫁接存活率？
你可以照這個流程做：
1. 嫁接前 1–2 週先養砧木。讓砧木水分充足、葉色正常，不要剛換盆、剛施重肥、剛受旱就嫁接。
2. 選溫暖穩定天氣。最好連續幾天無大雨、無寒流、無強風。溫度約 25–30°C 最理想。
3. 接穗先去葉保濕。剪下接穗後去除大葉，只留短葉柄，並用濕紙巾包住，避免失水。
4. 砧木與接穗粗細盡量相近
   粗細越接近，形成層越容易對準。
5. 刀具一定要利
   切面平滑，比塗很多藥更重要。
6. 包紮完整，切口不外露
   嫁接膜要覆蓋整個接合區，接穗頂端也要封住。
7. 嫁接後遮陰 2–4 週
   不要讓太陽直射接穗。接穗未發芽前，水分主要靠保濕，不是靠根部吸收。
8. 砧木萌芽要及時抹除
   嫁接處以下若長出新芽，要抹掉，避免養分被砧木芽搶走。
9. 接穗萌芽後不要太早拆膜
   看到發芽不代表完全癒合。通常等新梢展葉、接合處穩定後，再逐步放鬆或拆除包紮。
10. 先少量試接，不要一次全接
    第一次建議先接 5–10 株，觀察哪個時間、哪個方法、哪棵母樹接穗最容易成功。

## 黃金果繁殖，可否使用扦插方法育苗？
可以試驗扦插，但我不建議把它當成黃金果主要育苗方式。對黃金果而言，較可靠的繁殖路線仍是：
```
實生苗育砧木 → 再嫁接優良品種。
```
UF/IFAS 的黃金果栽培資料指出，黃金果最常用種子繁殖，也可將優良品種嫁接或芽接在實生砧木上；嫁接苗通常可比實生苗更早結果。 另一篇菲律賓黃金果研究[AskIFAS](https://ask.ifas.ufl.edu/publication/HS300)也顯示，黃金果以黃金果實生苗作砧木進行靠接與劈接可以成功，但其他砧木親和性較差。

#### 一、黃金果可否用扦插？
理論上可以嘗試，但成功率通常不穩定，不適合大量育苗。原因有三個：
- 第一，黃金果是木本熱帶果樹，不像番石榴、桑樹、火龍果那類容易發根。UF/IFAS 的熱帶果樹繁殖資料提醒，許多熱帶果樹不適合扦插，因為扦插苗可能形成較弱根系，日後容易支撐力不足或倒伏。
- 第二，黃金果若用扦插苗，通常沒有明顯主根，根系多為不定根。這對花蓮這種可能遇到強風、颱風、豪雨的地方不利；用實生苗當砧木，根系通常較深、支撐力較好。
- 第三，黃金果枝條切口有乳汁，扦插時容易因切口阻塞、失水、腐爛而失敗。這也是很多山欖科、乳汁較多的果樹繁殖較偏向嫁接、靠接，而不是單純扦插的原因。

#### 二、什麼情況下可以試扦插？
若你只是想做小規模實驗，可以試。但不要把所有材料都拿去扦插，建議只用少量枝條測試。較適合的扦插條件如下：
| 項目  | 建議                   |
| --- | -------------------- |
| 枝條  | 半木質化枝，不要太嫩也不要太老      |
| 長度  | 約 10–15 公分           |
| 葉片  | 剪掉大部分葉片，只留 1–2 片並剪半葉 |
| 介質  | 珍珠石＋泥炭土，或粗砂＋椰纖       |
| 環境  | 50–70% 遮陰，高濕度，通風     |
| 溫度  | 約 25–30°C            |
| 發根粉 | 可使用 IBA 發根劑          |
| 管理  | 介質濕潤但不可積水            |

操作上可把枝條下端斜切，稍微擦除乳汁，沾發根粉後插入乾淨介質約 3–5 公分，再用透明罩、育苗箱或噴霧設備維持濕度。重點是高濕但不悶爛。

#### 三、扦插苗的缺點
扦插苗即使發根，也有幾個問題：
- 根系較淺，支撐力較弱。
  這對田間果樹不是小問題，尤其黃金果未來會長成中大型果樹。
- 成活後初期長勢可能慢。
  發根成功不代表已經形成健全根系，移盆、下田時仍容易死亡。
- 容易腐爛。
  黃金果喜濕潤，但扦插時切口又怕濕悶，所以管理難度比種子育苗高。
- 沒有砧木優勢。
  嫁接苗可以利用實生砧木較強的根系，再接上品質穩定的接穗；扦插苗則根與枝都是同一母株，不一定適合你的田間土壤。

#### 四、若要提高扦插成功率
你可以這樣做：
1. 選春末到初夏，氣溫穩定時進行。
2. 選健康母樹的半成熟枝，不用太嫩的新梢。
3. 枝條剪下後馬上處理，避免失水。
4. 使用乾淨、排水好的無菌介質。
5. 下端沾 IBA 發根粉或發根液。
6. 保持空氣濕度，但介質不要長期濕爛。
7. 放在明亮散射光，不可全日照。
8. 發根前不要施肥。
9. 發根後先移小盆馴化，不要立刻下田。
10. 成活後觀察根系是否強壯，再決定是否保留。

#### 五、總結扦插的建議
若你的目標是種出健康田間黃金果樹，我不建議以扦插為主。比較穩的方式是：
```
黃金果種子育實生苗 → 養到鉛筆粗 → 嫁接優良母樹接穗 → 再下田定植。
```
扦插可以當作副試驗，但不要取代實生砧木與嫁接。尤其你是在花蓮地區，未來要面對風、雨、排水與土壤適應問題，有強健根系的實生砧木會比扦插苗更可靠。


## 黃金果繁殖，可否使用空中壓條／高壓法方法育苗？
使用空中壓條／高壓法繁殖黃金果，但我會把它定位為：可以少量試驗、保留母樹性狀，但不建議作為主要量產育苗法。
原因是黃金果屬於較難發根的果樹。農業部知識入口網回覆提到，不易扦插繁殖的作物包括黃金果，屬於較難發根作物；菲律賓的黃金果研究也指出，黃金果對 marcotting，也就是高壓／空中壓條，反應不佳，但用黃金果實生苗作砧木進行靠接與劈接可成功。
#### 結論先說：
黃金果可以試空中壓條，但成功率通常不如荔枝、龍眼、番石榴、蓮霧等容易高壓的果樹。
UF/IFAS 的黃金果研究資料也指出，黃金果最常用種子繁殖，也可以嫁接或芽接在實生砧木上；嫁接苗有機會比實生苗更早結果。
```
比較建議的繁殖順序仍是：
第一選擇：實生苗育砧木 → 嫁接優良母樹接穗。
第二選擇：靠接或劈接。
第三選擇：少量空中壓條試驗。
不建議：大量扦插或大量高壓作為商業苗來源。
```

#### 空中壓條的優點
空中壓條最大的優點是：可以複製母樹性狀。如果你有一棵黃金果母樹，果實品質好、甜度高、果形佳、產量穩定，用空中壓條繁殖出來的苗，理論上會與母樹基因相同。熱帶果樹繁殖資料指出，無性繁殖如扦插、空中壓條與嫁接，可保留母樹品種特性；空中壓條是在枝條仍連接母樹時誘導生根，因此比一般扦插較不容易立刻失水死亡。但問題是：會不會順利長根，是另一回事。

#### 空中壓條的缺點
黃金果高壓苗最大的缺點有三個。
- 第一，發根不穩定。黃金果不是容易發根的作物，可能包了 2–4 個月仍然只長癒傷組織，沒有足夠根系。農業部資料也把黃金果歸在較難發根作物。
- 第二，沒有主根。夏威夷大學熱帶果樹繁殖指南指出，扦插苗與空中壓條苗雖然與母樹基因相同，但沒有主根；嫁接在實生砧木上則可利用砧木主根，對土壤適應與強風固定較有利。 這一點對花蓮很重要，因為東部常有颱風、強風、豪雨，沒有主根的高壓苗日後比較需要支柱、防風與良好排水。
- 第三，下田後抗逆性可能較弱。高壓苗根系多集中在表層，初期對乾旱、積水、強風較敏感；若沒有先在盆內養根，直接下田風險較高。

#### 什麼情況可以試高壓？
- 你可以在這幾種情況下試：
  - 已有一棵果實品質很好的母樹；
  - 想保留該母樹品種；
  - 只是少量繁殖，不是大量育苗；
  - 有遮陰、保濕、育苗盆與後續馴化空間；
  - 可接受部分失敗。
- 不建議在這些情況下做：
  - 樹尚未結果，不知道果實品質；
  - 枝條太嫩或太老；
  - 冬季、寒流前、颱風季前；
  - 母樹正在缺水、落葉、病蟲害嚴重；
  - 想大量生產田間定植苗。

#### 黃金果空中壓條操作方法
- 選擇時間：
  避免冬季、寒流、連續豪雨與颱風季。黃金果適合溫暖潮濕環境，UF/IFAS 指出其最佳生長溫度約 20–35°C，且幼樹對低溫與強風敏感。
  以台灣環境來看，建議在：
  - 4–6 月春末初夏
  - 9 月初秋、颱風少且氣溫仍高時
- 選擇枝條：
  選母樹外圍、健康、半木質化枝條。不要選太嫩的新梢，也不要選太老、太粗、木質化嚴重的老枝。
  夏威夷大學的空中壓條說明也提到，通常在靠近枝梢的一段枝條上環剝，並以濕介質包覆誘根。
  建議條件：
  - 枝徑約 0.8–1.5 公分；
  - 枝條仍有活力，葉片正常；
  - 不是正在開花結果的枝；
  - 枝條位置方便包紮與日後剪下；
  - 距離枝梢約 30–60 公分處操作。
- 環剝
  在枝條上環剝一圈，寬度約 2–3 公分。把樹皮、韌皮部與形成層刮乾淨，露出木質部。這一步很重要。若形成層沒有刮乾淨，傷口可能又癒合回去，只長皮不長根。黃金果枝條可能會流乳汁，建議等乳汁稍微停止，或用乾淨紙巾輕輕擦掉，再包介質。不要把傷口泡水。
- 使用發根劑
  因黃金果較難發根，建議使用 IBA 發根劑。可塗在環剝上緣附近，也就是預期長根的位置。不要過量塗抹，也不要讓藥劑流到整個枝條上。重點是刺激傷口附近形成不定根。
- 包覆介質
  介質可用：水苔；椰纖；泥炭土；水苔＋珍珠石；椰纖＋少量珍珠石。
  介質要濕潤但不能滴水。太乾不發根，太濕會爛皮、發霉。
  包好後用透明塑膠布包住，再外包鋁箔或黑塑膠，避免日曬讓根區過熱。夏威夷大學的圖解也說明，空中壓條可用濕泥炭或介質包覆，再用塑膠固定保濕，外層可遮光。
- 等待發根
  黃金果不要期待很快。一般可能要 2–4 個月以上，甚至更久。期間每 2–3 週檢查一次即可，不要一直拆開看。
  判斷可以剪下的標準是：
  - 塑膠內可見白根或褐化新根；
  - 根量不是只有幾條，而是已包住介質一部分；
  - 枝條葉片仍正常，沒有嚴重萎凋；
  - 介質沒有腐臭或發黑爛掉。

#### 高壓苗剪下後怎麼處理？
這一步很關鍵。很多高壓不是死在樹上，而是死在剪下後。
等根系長滿盆、苗勢穩定後，才考慮下田。若在花蓮田間種植，建議下田後一定要立支柱、防風、覆蓋樹盤，避免颱風季前定植。
剪下後不要立刻下田。建議：
- 先剪掉部分葉片，減少蒸散；
- 保留高壓介質，不要拆散根球；
- 種入排水良好的盆土；
- 放在 50–70% 遮陰處；
- 保持濕潤但不積水；
- 至少養根 1–2 個月；
- 看到穩定新芽後再慢慢增加日照。

#### 高壓苗與嫁接苗比較
| 項目       | 空中壓條苗   | 實生砧木嫁接苗   |
| -------- | ------- | --------- |
| 是否保留母樹性狀 | 可以      | 可以        |
| 是否有主根    | 沒有明顯主根  | 有實生砧木主根   |
| 抗風性      | 較弱      | 較佳        |
| 土壤適應性    | 較弱      | 較佳        |
| 發根成功率    | 不穩定     | 嫁接熟練後較穩   |
| 適合用途     | 少量保種、試驗 | 田間栽培、量產育苗 |
| 對花蓮田間適應  | 要加強支撐防風 | 較推薦       |

#### 我的建議
若你只是想試驗，可以挑 3–5 條枝條做空中壓條，不要一次大量做。成功後先盆養，不要直接下田。
但若你的目標是建立田間黃金果園，我仍建議採用：
新鮮種子育實生苗 → 苗徑接近鉛筆粗 → 以優良母樹接穗劈接、腹接或靠接 → 養成嫁接苗後下田。
簡單判斷就是：空中壓條可試，但不是黃金果最穩的繁殖法；真正要種田間果樹，嫁接苗比高壓苗更可靠。
